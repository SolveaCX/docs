## 工作原理

Flatkey 位于你的应用与 AI 提供商之间。发送请求时：

1. 客户端使用 Flatkey API 密钥调用 `https://router.flatkey.ai/v1`。
2. Flatkey 验证请求，并从预付费余额中扣除费用。
3. 请求被转发到官方提供商 API（OpenAI、Anthropic 等）。
4. 响应直接返回客户端，不会转换内容。

你的提示词和模型回复从不被存储。Flatkey 只记录模型、Token 数量和时间戳等元数据，用于计费和分析。
