## 使用模型 ID

所有模型都使用厂商标准 ID。在请求的 `model` 字段中传入模型 ID：

```python python
response = client.chat.completions.create(
    model="gemini-2.5-flash",  # Google Gemini via Flatkey
    messages=[{"role": "user", "content": "Hello!"}],
)
```

包含全部版本和实时健康数据的完整列表请访问 [flatkey.ai/models](https://flatkey.ai/zh/models)。
