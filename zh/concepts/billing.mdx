Flatkey 完全采用预付费计费模式——先向余额充值，只有 API 请求实际使用模型时才会扣费。没有月度订阅、周期性收费或意外账单。每次充值都会立即获得赠送额度，且永不过期。

## 充值档位

| 充值金额 | 赠送额度 | 获得的总余额 | 实际节省 |
|---|---|---|---|
| $10 | +$3 | $13 | 30% |
| $20 | +$8 | $28 | 40% |
| $200 | +$100 | $300 | 50%（最划算） |
| 企业版 | 自定义 | 自定义 | 自定义 |

<Tip>
  赠送额度可与模型折扣叠加。模型本身价格已是官方标价的 60–90%，再叠加充值赠送后，实际成本最低可至官方价的 5 折。
</Tip>

## 费用如何计算

每个模型都有输入价格和输出价格，均按每百万 Token 计价。发送请求时：

1. Flatkey 在请求开始时估算 Token 费用。
2. 从余额中预留估算金额。
3. 响应完成后确认准确的 Token 数量。
4. 扣除准确费用，并退回多预留的部分。

[模型目录](/zh/reference/model-list)显示的是充值 $200 档位后的 **赠送后** 实际价格，同时也列出官方价格供参考。

## 费用计算示例

以充值 $200 后使用 `gpt-4o-mini` 为例：

- OpenAI 官方价格：每 100 万输入 Token $0.15
- Flatkey 标价：每 100 万输入 Token 约 $0.09（优惠 40%）
- 叠加 $200 充值赠送：每 100 万输入 Token 约 $0.06（相当于官方价约 5 折）

对于一个 1,000 Token 的请求：
- 官方价格：$0.00015
- Flatkey 实际价格：$0.00006

## 在控制台中查看余额和用量

前往[控制台](https://console.flatkey.ai/wallet?lng=zh)中的 **钱包**：

- 查看当前余额和赠送额度明细
- 发起新的充值
- 查看全部充值交易记录

前往 **使用日志**：

- 查看每次请求的模型、Token 数量和费用
- 按日期范围、模型或 API 密钥筛选
- 导出日志用于对账

## 企业计费

对于月度用量较大的团队，Flatkey 提供企业方案：

- 自定义月度用量和价格
- 账单结算，无需预付费充值
- 自定义路由折扣
- 团队采购支持和专属客户经理

如需了解企业条款，请联系 [support@flatkey.ai](mailto:support@flatkey.ai)。

## 退款与余额

除[退款政策](https://flatkey.ai/zh/refund-policy)涵盖的情况外，预付费余额不可退款。赠送额度会在充值后立即发放，并与付费额度适用相同退款条款。

<Note>
  余额永不过期。今天充值的额度可以无限期使用。
</Note>
